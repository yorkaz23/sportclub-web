# Documentación: Uso de Inteligencia Artificial

Para el desarrollo de este proyecto, se utilizó asistencia de IA como apoyo para la estructuración y resolución de problemas técnicos.

## ¿Cómo se utilizó?
1. **Generación de estructura base:** Se utilizó para traducir los bocetos iniciales (wireframes) a código HTML5 semántico.
2. **Sistema de Diseño (CSS):** Se solicitó apoyo para crear una hoja de estilos global (`styles.css`) que mantuviera la coherencia visual exigida por el cliente (colores corporativos y tipografía).
3. **Resolución de problemas de despliegue:** La IA fue clave como tutor para diagnosticar y solucionar errores de visualización en GitHub Pages (solución del error 404 modificando nombres de archivos y forzado de caché del navegador).

## Evidencia de comprensión y modificación
- Se aplicó la diferenciación visual en la navegación mediante clases CSS dinámicas (`--theme-user`, `--theme-coach`, `--theme-admin`) para cumplir los requisitos de roles.
- Se enlazaron manualmente los distintos módulos (Login, Registro, Dashboards) para generar un flujo de navegación coherente en un entorno estático.
- Se gestionó el repositorio en GitHub, solucionando conflictos de rutas de imágenes (logo) para el correcto funcionamiento en el entorno de producción web.
